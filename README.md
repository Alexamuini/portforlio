# Petero Mzee Portfolio Clone

A static local clone of the portfolio site at https://iam-petero.vercel.app/.

## Files
- index.html — homepage clone
- projects.html — featured projects page
- contact.html — contact page
- styles.css — shared styling

## Run locally
Open index.html in your browser, or use a simple local server:

```bash
cd /home/lafikih/Documents/zenith
python3 -m http.server 8000
```

Then visit http://localhost:8000.
